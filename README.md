LockstepFundation
===========================
一个基于Unity的简单的帧同步基础，包含定点数学库，基于四叉树的多边形的碰撞检测以及射线检测


特性
------
- 实现了基于定点数的数学库以及简单的物理检测
- 实现了基于2D平面的定制物理
- 预留帧事件接口，方便多人游戏接入
- 默认碰撞体可以沿静态物体四边平滑移动
- 数据与视图的分离

入门及示例
------
将整个工程直接使用Unity打开，其中MGF是核心文件与具体的业务无关，可以单独取出来使用，示例Demo场景存放在Demo/SampleScene.unity

使用版本：2018.3.2f1
